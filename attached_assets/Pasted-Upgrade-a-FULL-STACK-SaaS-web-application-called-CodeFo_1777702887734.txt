## 3. TEMPLATE ENGINE

* Support multiple templates:

  * Minimal
  * Cyberpunk
  * Corporate

* Dynamic rendering:

```js
const templateMap = {
  minimal: MinimalLayout,
  cyberpunk: CyberpunkLayout,
  corporate: CorporateLayout,
};

const Layout = templateMap[user.templateId] || DefaultLayout;
return <Layout data={userData} />;
```

* Allow:

  * Theme color selection
  * Dark/Light mode
  * Font customization

## 6. CONTACT FORM (REAL-TIME EMAIL)

* Visitors can send messages
* Messages go directly to portfolio owner's email

### Implementation:

* Use SendGrid or Resend (preferred over Nodemailer)
* Backend API:

  * POST /api/contact
* Payload:

  * name
  * email
  * message
  * ownerEmail

### Security:

* Do NOT expose user email publicly
* Add rate limiting
* Optional: reCAPTCHA
